# 🚀 EcomIQ – E-Commerce Customer Intelligence Dashboard

A full-stack data analytics dashboard built to analyze e-commerce performance, generate insights, and visualize key business metrics in real-time.

---

## 📊 Features

* 📈 Revenue Trend Analysis (with Forecast)
* 👥 Customer Segmentation (RFM-based)
* 🧠 AI-driven Insights Engine
* 🛒 Top Products Performance Tracking
* ⚡ Interactive Dashboard UI (Dark Theme)
* 🔄 Real-time API integration using FastAPI

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* Tailwind CSS
* JavaScript
* Chart.js

**Backend:**

* FastAPI
* Python
* Pandas
* NumPy

## ⚙️ Setup Instructions

### 1. Clone Repository

```
git clone https://github.com/YOUR_USERNAME/ecomiq-dashboard.git
cd ecomiq-dashboard
```

---

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

---

### 3. Install Dependencies

```
pip install -r requirements.txt
```

---

### 4. Run the Application

```
uvicorn backend.main:app --reload
```

---

### 5. Open in Browser

```
http://127.0.0.1:8000
```
